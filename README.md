# 🤖 BotMaker - منصة إنشاء بوتات تليجرام

منصة متكاملة لإنشاء وإدارة بوتات تليجرام احترافية بدون برمجة.

## ✨ الميزات

- ✅ **إنشاء بوتات** - إنشاء بوتات حقيقية عبر Token من @BotFather
- ✅ **ربط القنوات** - إضافة البوت كمشرف في قناتك والتحكم بها
- ✅ **الأوامر المخصصة** - إضافة أوامر مثل `/start`, `/help`, `/info`
- ✅ **الردود الآلية** - ردود تلقائية عند كتابة كلمات معينة
- ✅ **الإرسال الجماعي** - إرسال رسالة لكل المشتركين دفعة واحدة
- ✅ **إحصائيات** - متابعة المشتركين والرسائل
- ✅ **نظام اشتراكات** - خطط مجانية واحترافية

## 🚀 التثبيت السريع

```bash
# 1. فك الضغط
tar -xzvf botmaker-full.tar.gz

# 2. تثبيت المتطلبات
npm install
# أو
bun install

# 3. إعداد قاعدة البيانات
cp .env.example .env
npx prisma generate
npx prisma db push

# 4. تشغيل الموقع
npm run dev
# أو للإنتاج
npm run build
npm run start
```

## 📝 كيفية الاستخدام

1. **إنشاء حساب** - سجل في الموقع
2. **الحصول على Token** - من [@BotFather](https://t.me/BotFather) في تليجرام
3. **إنشاء بوت** - أدخل الـ Token في لوحة التحكم
4. **إضافة أوامر** - أنشئ أوامر وردود
5. **ربط بقناة** - أضف البوت كمشرف في قناتك
